2025.12.24 ✨ 대규모 업데이트 (v2.0)
  • [엔진 교체] Windows 내장 OCR에서 Windows Snipping Tool OCR 엔진(OneOCR)으로 핵심 OCR 엔진 교체

  • [웹훅 기능] 클라우드플레어 연동 시 디스코드 웹훅을 통해 링크를 전송시킬 수 있는 기능 추가
